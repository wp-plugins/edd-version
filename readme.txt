Shortcode to output the version number of an EDD download. Requires the EDD Software Licensing add-on.

== Description ==

When using the [Easy Digital Downloads](http://wordpress.org/extend/plugins/easy-digital-downloads/) plugin to sell digital downloads, this plugin will allow you to display the version number for each of your downloads.

**What does EDD Changelog do?**

It adds an `[edd_version]` shortcode you can use to display the software version number for EDD downloads.

== Installation ==

Extract the zip file and just drop the contents in the `wp-content/plugins/` directory of your WordPress installation and then activate the Plugin from Plugins page.
